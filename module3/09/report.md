## Задание 09 (Семафоры POSIX)
Родительский процесс генерирует строки (наборы из случайного количества случайных чисел) и записывает их в файл.
Дочерний процесс анализирует строки (если в файле есть необработанные строки): в каждой строке находит максимальное и
минимальное число и выводит эти значения на экран.
Доступ к файлу управляется семафором.  

#### Реализация
- Родительский процесс генерирует строку с числами и записывает в общий буфер (shared_data). После этого дочерний процесс обрабатывает строку. Всего таким образом генерируется и обрабатывается 7 строк. Для реализации сощдано три семафора: для каждого процесса и мьютекс для общих данных.
- Компиляция  
```
gcc semaphore.c
./a.out
```
